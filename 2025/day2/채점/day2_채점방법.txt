Day2 채점 방법 안내

채점 도구 사용법
1. CloudShell에 파일 업로드
Day2check.sh 파일을 CloudShell에 업로드
2. Bastion 서버로 파일 전송
scp -i "키.pem" Day2check.sh ec2-user@{bastion EIP}:/home/ec2-user/


3. 스크립트 실행
chmod +x Day2check.sh
./Day2check.sh


4. 결과 점수 확인
스크립트 실행 후 나오는 점수 확인

* 채점 구성 (총 60점 + 추가 5점)
자동 채점 (31점)
쉘 스크립트로 자동 확인되는 문항들
수동 채점 (29점)
다음 문항들은 채점기준표와 비교하여 수동 채점:

3-2, 3-3, 4-2, 4-3, 6-4, 7-2

**특별 주의사항**
6-3 Glue 크롤러 확인 (5점):

쉘 스크립트에서 통과해도 채점기준표 사진과 형식이 일치해야 온전한 5점 획득

추가 점수
7-2 최종문제: +5점 추가 점수 가능

✅ 최종 점수 계산
쉘 스크립트: 31점
수동 채점: 29점
총합: 60점
추가 점수: 최대 5점
